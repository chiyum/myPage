載入狀態 Document.readyState

Document.readyState為網頁的載入狀態。共分成三個狀態: loading / 載入 。document 仍在載入。 interactive / 互動。文件已經完成載入，文件已被解析， 但是諸如影象，樣式表和框架之類的子資源仍在載入。 complete / 完成。T文件和所有子資源已完成載入。 因此，當若要再載入時有載入畫面，可以在一開始就先顯示載入畫面， 接著在載入狀態為完成時再使載入畫面消失。作法為: document.onreadystatechange =function(){ let loading = document.querySelector('.planet') if(document.readyState == 'complete'){ loading.classList.add('active') } } 白話說明以上操作。首先是onreadystatechange這個程式碼， 此代碼在document.readyState的狀態改變時會觸發，所以我們設置了個if。 當載入狀態為完成時，使載入畫面加入active這個class，使畫面消失
